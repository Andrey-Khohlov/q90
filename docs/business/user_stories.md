# User Stories 
В формате: "Как [роль], я хочу [действие], чтобы [ценность]"

- Как любитель кофе, я хочу вести дневник заварок, чтобы запоминать удачные рецепты
- Как любитель кофе, я хочу получать оценку вкуса кофе для его покупки
- Как любитель кофе, я хочу видеть описания вкуса кофе пользователями, которым я доверяю
- Как фермер, я хочу видеть оценки моего кофе, чтобы понимать предпочтения рынка
- Как модератор, я хочу получать уведомления о жалобах, чтобы быстро чистить контент
- Как анонимный пользователь, я хочу смотреть каталог, чтобы решить, стоит ли регистрироваться
